Getting Started

In this section we aim to provide a quick introduction for each module available in the Fisheries Trade Tools. The short videos below show the primary features developed in order to the user be able to interact accross the modules.

Module: Getting Data

It shows how the user can build the data for each country wished. After the selction, the user must save the data shown, otherwise the data will be discarded.

Steps:

  1. Select a country from the list.
  2. Select a year.
  3. Select a data source to imports data.
  4. Select a data source to exports data.
  5. Click on the button “Build data”.
  6. As the data has been loaded the user must click on “Save”.

Module: Mapping

Run Mapping

  1. Select the year;
  2. Select the country;
  3. Click on the button “Run Mapping”

Discard mapping

  1. Select the row to be discarded.
  2. Click on the “ Discard”;
  3. The user must “ save after discarding the row.

Add a new mapping

  1. Click on “ Add mapping”
  2. Fill out all fields required: Tariff Line; ISSCFC code or ISSCFC ID; start year; end year, HS6 use.
  3. Click on “OK”
  4. For seeing the update the user should browser around the options under the combo box “Filters”, and then go back to the option “All”.

Update mapping

Any change in the mapping table should be save, otherwise all alteration made will be lost. The save function creates a working table which is not considered in the process until to be updated. It works like a temporary table. Once that the user updates the mappping table, i.e., export all changes to the production table, then all modifications will be considered to the next steps.

  1. After any change the user should click on “Save”.
  2. Click on the “Update”
  3. The user must go back to the tab “Run mapping” and run again the mapping process. It is important, because the script will perform some test over the mapping table and also in the trade data.
